## The problem it solves

Modern LLMs can produce functional web pages from a simple prompt. But their output converges on the same aesthetic: Inter font, blue primary buttons, purple-pink gradients, large-radius cards, emoji as icons, and fabricated testimonials. Technically correct, visually generic.

Web Design Skill injects **design taste** into the AI's decision-making process — forcing it to declare a coherent design system, avoid overused patterns, and derive colors with perceptual precision before writing a single line of code.

## Getting started

<Steps>
  <Step title="Copy the skill directory">
    Place `.agents/skills/web-design-engineer/` (containing `SKILL.md` and `references/advanced-patterns.md`) into your project root.
  </Step>
  <Step title="Start your AI agent">
    Open Claude Code, Cursor, or your preferred agent in the project directory. The agent picks up the skill automatically when your request involves visual front-end work.
  </Step>
  <Step title="Request a visual artifact">
    Ask for a web page, slide deck, dashboard, or prototype. The skill activates and guides the agent through the six-step design workflow.
  </Step>
  <Step title="Review the design declaration">
    Before writing code, the agent declares its color palette, typography, spacing, and motion choices. Approve or redirect before a single line is written.
  </Step>
</Steps>

<Note>
  Some tools use `.claude/skills/` instead of `.agents/skills/`. The skill files are identical — place them in whichever directory your agent expects.
</Note>
